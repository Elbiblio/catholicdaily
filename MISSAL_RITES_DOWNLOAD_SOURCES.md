# Missal Rites Download Sources

This document catalogs online sources for downloading complete missal rites (collect prayers, communion prayers, etc.) for the target languages.

## Copyright Warning

**IMPORTANT:** Most official missal translations are copyrighted by their respective episcopal conferences. Before using any downloaded content, you must:
1. Verify the copyright status
2. Check if permission is needed for non-commercial use
3. Consider fair use for educational/devotional purposes
4. Contact the episcopal conference for licensing if needed

4. **Missale Meum API** - Most comprehensive API for Extraordinary Form (1962 Missal), "free to use" but verify commercial terms

**Best for Extraordinary Form (1962 Missal):**

2. **SanctaMissaAPI** - 1962 Missal propers via API (verify license)

**Best for Historical/Traditional Texts:**

**Key Findings:**
- **Liturgia Diária API** appears to provide Portuguese collect prayers ("Oração do Dia") - this is the only open source API found that actually provides Ordinary Form (Vatican II) missal prayers (collect), not just readings/calendar. However, license is unclear and must be verified.
- **Missale Meum API** is the most comprehensive API found for Extraordinary Form (1962 Missal), with full propers, ordo, prayers, chants, and votive Masses in English/Polish. "Free to use" but commercial terms need verification.
- **lectionary-js** is MIT licensed but Lutheran-based, not Catholic.
- Most comprehensive sources are for Extraordinary Form (1962), not Ordinary Form (Vatican II).

### Why No Open Source Database?

**Liturgical Norms (Flexibility):**
- GIRM 69-71 states intentions should be "sober" and "composed with a wise liberty"
- Intentions are meant to be adapted to local circumstances, current events, and community needs
- The Church allows flexibility for local bishops, pastors, and communities

**Commercial Reality:**
- While standard collections exist (Years A, B, C), they are commercial products
- Major publishers (Liturgical Press, GIA) invest in creating these resources
- No incentive to make them open source as they are revenue-generating

**Practical Implications:**
- Unlike readings (which are fixed in lectionaries), intercessions CAN be standardized but ARE typically sold as commercial books
- There IS a standard calendar-based structure, but it's behind copyright/paywalls
